# navigator_exam

５つのページを [go_router](https://pub.dev/packages/go_router) を使用して遷移するデモ。
bottom navigation barのメニュー間の移動と引数を持つページへの移動。

### pages:
  - tab1 (initial page)
    - sub page1
    - sub page2
  - tab2
  - tab3

### to launch 
1. flutter pub get
2. flutter run
